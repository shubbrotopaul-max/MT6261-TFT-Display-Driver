# MT6261 TFT Display Library

Driver library for MT6261 based TFT displays.

Compatible with:
- ESP32
- Arduino

## Features

- Fast SPI display driver
- Graphics drawing
- Clock demo
- UI support

## Display

2.8 & 2.4 inch TFT  
Controller: MT6261

## Pin Connection

| Display | ESP32 |
|-------|------|
| VCC | 3.3V |
| GND | GND |
| CLK | 18 |
| MOSI | 23 |
| CS | 5 |
| RST | 4 |
